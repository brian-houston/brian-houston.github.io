Winding Circles

Reference

Give a list operations where each operation is on one line.

Format for operations is shown below.

Rotation and scale operations follow sine function.

Offset is zero by default.

Color determined by winding number.

Circle Operation
- C (frequency) (radius) (offset)
Rotation Operation
- R (frequency) (midline) (magnitude) (offset)
Scale Operation
- S (frequency) (midline) (magnitude) (offset)
